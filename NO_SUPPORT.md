# No Support / As-Is Notice

This repository is shared as a public reference and starting point.

It was created for a specific one-time photo/video archive cleanup workflow. It is not intended to be a polished commercial product or an actively maintained open-source project.

## What this means

- No support is promised.
- Issues may not be answered.
- Pull requests may not be reviewed.
- Bugs may exist.
- Behavior may need to be adapted for your archive, operating system, codecs, metadata, folder structure, or tool versions.
- You should test on copies of files before using it on a real archive.

## Recommended use

Fork the repository, test it on a small copied folder, and modify it for your own needs.
